What is FIRELab?

Every year, specially during summer when the temperatures are higher, there're several forest fires everywhere, including our country.

The imagery captured using UAV provides important data to develop studies on the spread of fires and the emission of polluting gases into the atmosphere.
However, the data is currently being processed and analysed manually, using non user-friendly commands.

FIRELab solves this problem by transforming raw drone imagery in good quality georeferenced data, ready to be used by environmental researchers and engineers.

Features

Area of Interest

We transform raw videos and images in orthophotomaps. The area is then divided into cells and the vegetation is characterized using smart methods to minimize user input.

Responsive Design

One of the biggest problems we aim to solve. Before, there was no interface, and the user had to do everything by himself, using only the command line.

Spatio-temporal Data

We provide accurate data that shows the fire's evolution over time. The burnt area, or the line that defines the front of the fire at a given moment are also modeled
